# Techcare System

## Nombre del proyecto
Techcare Systems - Sistema Integral de Supervisión y Regulación Tecnológica

## Nombre del estudiante
Zandra Mareva Romero Villacis

## Objetivo del sistema
Desarrollar un sistema que permita registrar y analizar el tiempo de uso de dispositivos tecnológicos para promover hábitos digitales saludables.

## Descripción de funcionalidades
- Registrar usuarios.
- Registrar horas de uso.
- Consultar historial.
- Buscar usuarios registrados.
- Mostrar estadísticas del uso.
- Generar recomendaciones según las horas registradas.
